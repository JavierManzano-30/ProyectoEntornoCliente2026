# Sistema de Aprobaciones (BPM)

## Objetivo
Describir como el modulo BPM gestiona aprobaciones de otros modulos.

## Entidad aprobaciones (propuesta)
- Campos sugeridos: modulo, registro_id, tipo, estado, aprobador_id, created_at, updated_at.

## Flujos cubiertos
- Ausencias (RRHH)
- Facturas (ERP)
- Presupuestos (ALM)
