# Metricas BI

## Objetivo
Consolidar las metricas expuestas por cada modulo para BI.

## Metricas por modulo (borrador)
- RRHH: empleados activos, tasa de rotacion, ausencias
- CRM: oportunidades en pipeline, tasa de conversion
- ALM: proyectos activos, velocidad de equipo
- Soporte: tickets abiertos, tiempo de resolucion
- ERP: valor de inventario, facturas pendientes
- BPM: aprobaciones pendientes, tiempo de aprobacion
