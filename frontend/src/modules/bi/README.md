# Módulo BI (Business Intelligence)

Este módulo contiene la lógica, componentes y pantallas para la visualización, análisis y generación de informes de datos empresariales.

- Dashboards interactivos
- Catálogo de informes
- Visualización de KPIs y métricas
- Gestión de datasets y alertas

Estructura basada en los módulos CRM, RRHH y Soporte.
