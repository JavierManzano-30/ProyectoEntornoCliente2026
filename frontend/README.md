# Frontend - Sistema ERP Modular

Sistema frontend desarrollado con React + Vite para la gestión integral de una empresa, con diseño profesional estilo Factorial.

## ��� Tecnologías

- **React 19** - Biblioteca de interfaz de usuario
- **Vite 7** - Build tool y dev server
- **React Router DOM** - Navegación entre páginas
- **Axios** - Cliente HTTP para peticiones a la API
- **Lucide React** - Iconos modernos
- **CSS Modules** - Estilos con scope

## ��� Instalación

```bash
# Instalar dependencias
npm install

# Copiar archivo de configuración
cp .env.example .env

# Configurar la URL de la API en .env
VITE_API_BASE_URL=http://localhost:3000/api
```

## ��� Ejecución

```bash
# Modo desarrollo
npm run dev

# Build para producción
npm run build

# Preview de producción
npm run preview

# Linting
npm run lint
```
